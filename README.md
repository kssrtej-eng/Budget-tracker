# Budget-tracker
Track your money like a pro with this minimalist Budget Tracker App built in Python!

This fully interactive app lets you:

Input your income and expenses

Categorize entries: Food, Transport, Entertainment, Bills, etc.

View monthly summaries with total income, expenses, and net balance

See bar/pie charts for quick visual insights

Clear all data or remove individual entries

Design: Minimalist, professional, white background with sleek black text. Perfect for portfolios, personal finance tracking, or client-ready demos.

Optional Advanced Features:

Export data as CSV

Sort entries by date or category

Fully responsive for desktop and mobile
